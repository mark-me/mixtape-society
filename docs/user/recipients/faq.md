## üéµ Playback & Features

### Why isn't there a download button?

**By design.** Mixtapes are for streaming only.

**Reasons:**
- Protects creator's music library
- Respects copyright
- Keeps files on secure server
- Prevents unauthorized distribution

**You can:**
- Listen unlimited times
- Bookmark for easy access
- Cast to better speakers
- Control quality settings

### Can I listen offline?

**Currently, no.** You need an internet connection to stream.

**Future possibilities:**
- Offline mode may be added later
- Would require creator permission
- Not available yet

**For now:**
- Use WiFi when possible
- Choose lower quality to save data
- Download music legally from other sources

### How much data does streaming use?

**Per song (approximate):**

| Quality | Per Song | Per Hour |
|---------|----------|----------|
| **Original** | ~10MB | ~150MB |
| **High (256k)** | ~7MB | ~105MB |
| **Medium (192k)** | ~5MB | ~75MB |
| **Low (128k)** | ~3MB | ~45MB |

**Default:** Medium quality (good balance)

**Tips:**
- Use WiFi when available
- Switch to Low on cellular
- Monitor your data usage
- Consider unlimited data plan

### Why does playback keep stopping?

**Common causes:**

**Slow connection:**
- Switch to lower quality
- Move closer to WiFi router
- Close other apps using internet
- Pause and let it buffer

**Browser issues:**
- Refresh the page
- Clear browser cache
- Update browser
- Try different browser

**Device issues:**
- Close other apps
- Free up RAM
- Restart your device
- Check available storage

**Server issues:**
- Creator's server might be slow
- Many people listening at once
- Contact creator about issues

### Can I adjust the volume?

**Yes!** Multiple volume controls:

**In the player:**
- Use the volume slider
- Click speaker icon to mute/unmute

**On your device:**
- Use device volume buttons
- System volume controls

**When casting:**
- Control volume on the cast device
- Or use the app's cast volume

**Note:** Player volume and device volume work together‚Äîboth affect final output.

## üì∫ Casting & Streaming

### How do I cast to my TV?

**Chromecast (Android/Chrome):**
1. Open mixtape in Chrome
2. Look for cast icon (üì°)
3. Tap and select your TV

1. Open mixtape in Safari
2. Start playing
3. Tap AirPlay icon
4. Select Apple TV
5. [Full guide ‚Üí](streaming.md#airplay-apple-devices)

### Can I use AirPlay?

**Yes!** For iPhone, iPad, and Mac users:

**Requirements:**
- Must use Safari browser (not Chrome)
- Apple TV, HomePod, or AirPlay 2 speakers
- Same WiFi network

**How to use:**
- Tap AirPlay icon in player
- Or use Control Center
- Select your device
- [AirPlay guide ‚Üí](streaming.md#airplay-apple-devices)

### Can I use Chromecast?

**Yes!** For Android, Chrome, and Windows:

**Requirements:**
- Chromecast device
- Chrome, Edge, or Opera browser
- Same WiFi network

**How to use:**
- Look for cast icon (üì°)
- Tap to select device
- Start streaming
- [Chromecast guide ‚Üí](streaming.md#chromecast)

### Why isn't AirPlay working?

**Common issues:**

**Wrong browser:**
- Must use Safari on iOS/Mac
- Chrome doesn't support AirPlay
- Switch to Safari

**Not on same network:**
- Phone and Apple TV need same WiFi
- Check WiFi settings
- Restart router if needed

**AirPlay disabled:**
- Check Apple TV settings
- Enable AirPlay
- Update Apple TV software

**Device issues:**
- Restart iPhone/iPad
- Restart Apple TV
- Update iOS/tvOS

### Can multiple people control playback?

**Depends on casting method:**

**Chromecast:**
- ‚úÖ Anyone on same WiFi can control
- ‚úÖ See what's playing
- ‚úÖ Change tracks or volume

**AirPlay:**
- ‚ùå Only device that started AirPlay
- Control from that device only
- Others need to stop and start their own

**Direct playback:**
- Only person with the browser open
- Not shared control

## ‚ö†Ô∏è Troubleshooting

### The player looks broken or glitchy

**Solutions:**

**Update browser:**
```
Check browser settings ‚Üí About
Update to latest version
```

**Clear cache:**
1. Browser settings
2. Clear browsing data
3. Reload mixtape

**Try different browser:**
- Download Chrome or Firefox
- Test in that browser

**Disable extensions:**
- Ad blockers might interfere
- Try incognito/private mode

### I hear crackling or distortion

**Audio quality:**
- Try different quality setting
- Higher might fix digital issues
- Lower might fix buffering

**Connection:**
- Slow internet causes artifacts
- Switch to lower quality
- Use WiFi instead of cellular

**Device:**
- Check headphone connection
- Try different headphones
- Restart your device

**File issue:**
- Might be the source file
- Contact creator
- Try other tracks

### Controls aren't responding

**Try:**
1. Refresh the page
2. Close and reopen tab
3. Restart browser
4. Restart device
5. Try different browser

**Check:**
- JavaScript is enabled
- No browser errors (F12 console)
- Browser is updated

### Battery drains quickly

**Why this happens:**
- Streaming uses network constantly
- Screen might stay on
- Processing audio

**To improve:**
- Lower quality setting
- Close other apps
- Reduce screen brightness
- Use battery saver mode
- Plug in while listening

## üí° Tips & Tricks

### How can I get the best audio quality?

**1. Use good headphones**
- Wired often better than Bluetooth
- Proper fit matters
- Over-ear vs in-ear preference

**2. Choose right quality**
- Original/High on WiFi
- Good headphones reveal difference
- Lower quality for casual listening

**3. Stable connection**
- WiFi over cellular
- Close to router
- Avoid congestion

**4. Cast to quality speakers**
- Chromecast to home system
- AirPlay to HomePod
- Better than phone speakers

### Can I make my own mixtapes?

**As a recipient, no.** You'd need:
- Your own music library
- A server to run Mixtape Society
- Technical setup knowledge

**Your options:**
1. Ask the creator for access to their server
2. Set up your own server
3. Use other services for playlists

[Learn about becoming a creator ‚Üí](../creators/index.md)
