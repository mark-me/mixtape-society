    üéØ Common Questions¬∂

    Do I need to create an account?¬∂

    No! Just click the link and start listening. That's it.

    Does it work on my phone?¬∂

    Yes! Works on any device with a modern web browser:

    • iPhone & iPad (Safari works best)
    • Android phones & tablets
    • Windows, Mac, Linux computers
    • Chromebooks

    Can I download the songs?¬∂

    No, streaming only. This protects the creator's music library and respects copyright. But you can listen as many times as you want!

    Will it use a lot of data?¬∂

    Mixtapes use about:

    • Original quality: ~10MB per song
    • High quality: ~7MB per song
    • Medium quality: ~5MB per song (default)
    • Low quality: ~3MB per song

    You can adjust quality in the player settings.

    Can I listen offline?¬∂

    Not currently. You need an internet connection to stream mixtapes.

    Can I cast to my TV or speakers?¬∂

    Yes! If you have:

    • Chromecast (Android/Chrome/Windows)
    • AirPlay device (iPhone/iPad/Mac)
    • Android Auto (in-car)

    2. Point at QR code

    4. Mixtape opens in browser

    üîä Audio Quality Guide¬∂

    The player defaults to Medium (192k) for balanced quality and data usage.

    When to adjust:

    Use Original/High if:

    • On WiFi
    • Have unlimited data
    • Want best quality
    • Using good headphones/speakers

    Use Medium if:

    • Normal mobile data
    • Balanced needs (default)
    • Standard headphones

    Use Low if:

    • Limited data plan
    • Slow connection
    • Playback keeps buffering
    • On cellular in remote areas
